# Endpoints — Productos

## GET /productos

### Response
**Query Params**  
- page (number) → página actual (default: 1)
- limit (number) → cantidad de productos por página (default: 20)
### Request
#### Ejemplo
```json
GET /producto?page=1&limit=20
```
### Response
<Tabs> <TabItem value="200" label="200">
```json
{
  "page": 1,
  "limit": 20,
  "total": 20,
  "data": [
    {
      "id_producto": "uuid",
      "nombre": "Serum Vitamina C",
      "imagen_producto": "https://...",
      "tipo": "skincare",
      "colores": ["#E2C9AA", "#F4D8BE"]
    }
  ]
}
```
</TabItem> <TabItem value="401" label="404/500">
```json
{
  "statusCode": 500,
  "message": "Error obteniendo productos"
}
```
</TabItem> </Tabs> 
## GET /productos/search/:name
**Params**
- name (string) → término a buscar

**Query Params**
- page (number) → default 1
- limit (number) → default 20

